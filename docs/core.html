Girdiler ve Çıktılar¶

Girdileri alıp çıktıları göndermek için Core sınıfını kullanmalısınız.

Örneğin modeliniz ses seviyesi verisine göre motoru açıp kapatıyorsa bunun gibi birşey yapabilirsiniz:

# ses seviyesini al
ses_seviyesi = await core.get_sound_level()

# Kendi fonksiyonumuzla motorun
çok_sesli_mi = karar_ver(ses_seviyesi)

# Çok sesli ise motoru aç/kapat
if çok_sesli_mi:
    await core.turn_on_motor()
else:
    await core.turn_off_motor()

# Robotun ekranına durumumuzu yazdır
if çok_sesli_mi:
    durum = "Çok sesli"
else:
    durum = "Normal ses düzeyi"

core.set_state("Ortam sesi: " + durum)
class robo_core.core.Core¶
async get_air_quality() → float¶

Hava kalitesi sensörünün algıladığı hava kalitesini AQI cinsinden döndürür.

async get_gas_amount() → float¶

Gaz sensörünün algıladığı gaz miktarını ppm cinsinden döndürür.

async get_humidity() → float¶

Nem sensörünün aldığı nem değerini yüzde cinsinden döndürür. (0-100)

async get_light() → float¶

LDR sensörünün algıladığı ışık miktarını lümen cinsinden döndürür.

async get_proximity() → float¶

Yakın mesafe sensörünün algıladığı mesafeyi cm cinsinden döndürür.

async get_pulse() → float¶

Nabız sensörünün ölçtüğü nabız değerini BPM cinsinden döndürür.

async get_rain() → float¶

Yağmur sensörünün algıladığı yağmur miktarını yüzde cinsinden döndürür. (0-100)

async get_sound_level() → float¶

Mikrofonun algıladığı ses seviyesini desibel cinsinden döndürür.

async get_temperature() → float¶

Sıcaklık sensörünün aldığı sıcaklık değerini santigrat cinsinden döndürür.

async get_ultrasonic_distance() → float¶

Ultrasonik mesafe sensörü ile ölçülen mesafeyi cm cinsinden döndürür.

async is_motor_on() → bool¶

Motorun açık olup olmadığını döndürür.

async send_message(message: str) → None¶

Robot ekranındaki terminale mesaj gönderir. Terminaldeki mesajlar, aşağı doğru kayar. Eğer yapay zekanızın sonucunu göstermek istiyorsanız bunun yerine set_state fonksiyonu kullanabilirsiniz.

async set_motor_angle(deg: int) → None¶

Servo motorun derecesini ayarlar.

async set_state(state: str) → None¶

Yapay zekanızın durumunu günceller. Robotun monitöründe herkesin yapay zekasının durumu gösterilir. Bu fonksiyonu çağırdığınızda önceki durumunuz silinir ve yeni durumunuz gösterilir.

Örnek: `python forecast = "Güneşli" // Yapay zekanın tahmin etiği hava durumu await set_state("Hava durumu: " + forecast) `
